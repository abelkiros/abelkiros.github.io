# abelkiros.github.io

Source for my personal site: **[abelkiros.github.io](https://abelkiros.github.io/)**

A single-file portfolio covering my work, writing, and journey into cybersecurity. Built from scratch — no frameworks, no build step, just HTML and CSS.

## About me

I'm Abel Mengesha Kiros — a Customer Technology Support Technician at the University of Rochester Medical Center, working overnight Tier 2 support across Strong Memorial and Highland Hospitals. I'm pursuing a B.S. in Cybersecurity and Information Assurance at WGU, and my long-term goal is to move into a cybersecurity engineer role.

The site covers my experience, certifications, projects, and the personal journey behind why I'm doing this work.

## Stack

- Single-file `index.html` (HTML + inline CSS + vanilla JS)
- Google Fonts: Instrument Serif, IBM Plex Sans, JetBrains Mono
- Hosted on GitHub Pages
- Dark mode default, light mode toggle, persisted in `localStorage`
